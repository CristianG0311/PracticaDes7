# PracticaDes7

Problema 1 - Suma (suma.php)
Se declaran dos variables numéricas y se obtiene su suma utilizando el operador +. El resultado se muestra en pantalla usando echo.

Problema 2 - Resta (resta.php)
Se declaran dos variables numéricas y se obtiene la diferencia entre ellas utilizando el operador -.

Problema 3 - Multiplicación (multiplicacion.php)
Se declaran dos variables numéricas y se obtiene su producto utilizando el operador *.
